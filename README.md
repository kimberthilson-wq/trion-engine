# trion-engine

    > Deterministic validation architecture for AI-assisted trading strategies.
    > AI suggests. Risk protects. Humans execute.

    Architecture

    - Multi-agent strategy design — AI helps design, validates via DSL
    - Redis Streams — Signal pipeline & runtime state management
    - Deterministic Validator — Every signal validated before execution
    - Strategy DSL — Human-readable strategy definition language
    - Backtesting Engine — Deterministic backtesting on real historical data

    Core Principles

    1. Transparency — Every decision is traceable
    2. Determinism — Same inputs always produce same outputs
    3. Human-supervised — AI never executes trades autonomously
    4. Risk-first — Risk engine always overrides AI signals
    5. Open infrastructure — Building in public

    What This Is NOT

    - ❌ Autonomous trading bot
    - ❌ Signal-selling platform
    - ❌ "Get rich quick" system
    - ❌ Black-box AI

    What This Is

    - ✅ AI-assisted strategy design
    - ✅ Deterministic execution validation
    - ✅ Professional trading workstation
    - ✅ Open-source infrastructure

    Tech Stack

    - Python / FastAPI (Backend)
    - React (Frontend)
    - Redis (Signal Pipeline)
    - Custom DSL (Strategy Definition)
    - PostgreSQL (Strategy Storage)
